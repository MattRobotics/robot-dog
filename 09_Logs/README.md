## Current Status

The quadruped platform is currently based on:

* 12x ST3215 servos
* Waveshare Bus Servo Adapter
* NormaCore Station
* Custom servo distribution board

Bus communication and hardware validation have been successfully completed.

The next development phase focuses on:

1. Stand pose
2. Leg IK
3. Gait generation
4. First walking tests
